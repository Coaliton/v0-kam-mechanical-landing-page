# Asset Rules

- All filenames must be lowercase.
- Use hyphens only (`-`), never underscores or spaces.
- Use descriptive filenames that match the content.
- Use `.jpg` for photos, `.png` for logos, and `.svg` for icons.
- Reference files in `/public` using absolute paths like `/images/folder/file.ext`.
- Always use Next.js `next/image` for rendering images.
